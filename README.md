# Welcome to your GitHub 101 activities

Static site for giving teachers and students the assignments for GitHub 101 sessions. 

available at [learnwithpj.com](https://learnwithpj.com)

## Current Activities
## Activities

1. **[Profile README](pages/profile-readme.md)** — Create a special GitHub repository that displays a personal README on your profile. Learners practice Markdown, branching, committing, and opening a pull request — all in one solo activity.

2. **[Collaborative Activity](pages/collaborative.md)** — A group activity (3–4 people) where each person creates their own branch and Markdown file in a shared repo, then opens, reviews, and merges pull requests — without conflicts.

3. **[Merge Conflict](pages/merge_conflict.md)** — Groups intentionally trigger a merge conflict by editing the same file on separate branches, then practice resolving it together using GitHub's conflict editor.

4. **[Using Issues and Projects](pages/projects-issues.md)** — A solo project management activity where learners create a repo, set up a Kanban project board, define milestones and custom labels, and create and track issues — simulating how a real dev team stays organized.

5. **[Working Locally](pages/working_locally.md)** — Learners install Git, GitHub Desktop, and VS Code, authenticate with GitHub, and run through the full local workflow: creating branches, committing changes, and pushing to GitHub, finishing by cloning their Profile README repo.
