# Welcome to GitHub 101 Activities

A collection of hands-on activities to help teachers and students learn GitHub fundamentals.

---

## Getting Started

These activities are designed to be completed in sequence, building your Git and GitHub skills step by step. Some of them are solo activities and some are dseigned for group work. 

### What You'll Learn

- Creating and managing repositories
- Making commits and tracking changes
- Working with branches
- Collaborating through pull requests
- Using GitHub Issues for project management

---

## For Teachers

These materials are free to use and adapt for your classroom.
