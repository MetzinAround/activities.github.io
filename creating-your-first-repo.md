# Creating Your First Repository

Learn how to create a new repository on GitHub and set it up for your project.

---

## Objectives

By the end of this activity, you will be able to:
- Create a new repository on GitHub
- Initialize the repository with a README
- Understand repository settings and visibility options

---

## Prerequisites

- A GitHub account ([sign up here](https://github.com/signup) if you don't have one)
- Basic understanding of what version control is

---

## Instructions

### Step 1: Navigate to GitHub

1. Go to [github.com](https://github.com) and sign in to your account
2. Click the **+** icon in the top-right corner
3. Select **New repository**

### Step 2: Configure Your Repository

1. **Repository name**: Enter a name for your repository (e.g., `my-first-repo`)
2. **Description** (optional): Add a brief description of your project
3. **Visibility**: Choose between:
   - **Public**: Anyone can see this repository
   - **Private**: Only you and collaborators can access it

### Step 3: Initialize the Repository

1. Check the box for **Add a README file**
2. Optionally select a **.gitignore** template for your project type
3. Optionally choose a **license**

### Step 4: Create the Repository

Click the green **Create repository** button.

## What's Next?

Now that you have a repository, you can:
- Edit the README to describe your project
- Add files to your repository
- Clone the repository to your local machine

---

## Challenges

Try these optional extensions:

1. **Customize your README**: Add sections like "About", "Installation", and "Usage"
2. **Explore settings**: Find where to add collaborators or change visibility
3. **Create another repository**: Practice makes perfect!
